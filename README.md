# Data Cleaning & Visualization Project

## Project Overview

This project demonstrates the complete data analysis workflow using the Titanic dataset. The objective was to clean, preprocess, analyze, and visualize raw data to uncover meaningful insights and improve data quality.

The project focuses on handling missing values, removing duplicates, detecting outliers, and creating visualizations that help in understanding patterns and trends within the dataset.

---

## Objectives

* Clean and preprocess raw data
* Handle missing values and duplicate records
* Detect and analyze outliers
* Perform Exploratory Data Analysis (EDA)
* Create visualizations for data storytelling
* Generate actionable insights from data

---

## Dataset

Dataset Used: Titanic Passenger Dataset

The dataset contains information about passengers aboard the Titanic, including demographic details, ticket information, passenger class, fare, and survival status.

Key Features:

* Passenger ID
* Passenger Class
* Name
* Gender
* Age
* Fare
* Embarkation Port
* Survival Status

## Data Cleaning Process

### Missing Value Treatment

* Missing values in the Age column were filled using the median value.
* Missing values in the Embarked column were filled using the most frequent category.
* Missing values in the Cabin column were replaced with "Unknown".

### Duplicate Removal

* Duplicate records were identified and removed from the dataset.

### Outlier Analysis

* Boxplots were used to identify outliers in the Fare column.

## Key Findings

* Female passengers had a significantly higher survival rate than male passengers.
* First-class passengers showed better survival chances compared to lower classes.
* Most passengers belonged to the third passenger class.
* Age distribution indicated a wide range of passenger demographics.
* Fare and passenger class exhibited a noticeable relationship.
* Certain embarkation ports contributed a larger proportion of passengers.
